# 🚀 Quick Start Guide

## AI-Powered Nepali & Sinhala Literature Translator

### 📁 Project Location
This project is located on your Desktop at:
```
C:\Users\user5\Desktop\AI-Nepali-Sinhala-Translator
```

### ⚡ Quick Commands

1. **Navigate to project:**
   ```bash
   cd "C:\Users\user5\Desktop\AI-Nepali-Sinhala-Translator"
   ```

2. **Start development server:**
   ```bash
   npm run dev
   ```

3. **Open in browser:**
   ```
   http://localhost:5173
   ```

4. **Build for production:**
   ```bash
   npm run build
   ```

### 🎯 Demo Features

✅ **Dashboard** - File upload with drag & drop  
✅ **Document Viewer** - Three-panel editor (Original, OCR, Translation)  
✅ **Manual Corrections** - Edit OCR and translation text  
✅ **Auto-save** - Offline storage using LocalStorage  
✅ **Export Options** - PDF, Text, Word formats  
✅ **History Management** - Track all processed documents  
✅ **Dark/Light Mode** - Theme switching  
✅ **Responsive Design** - Works on all screen sizes

### 🆘 Troubleshooting

**If the server won't start:**
1. Make sure you're in the correct directory
2. Run `npm install` again
3. Check if port 5173 is available

**If dependencies fail:**
1. Delete `node_modules` folder
2. Run `npm install` again
3. Try `npm install --legacy-peer-deps`
