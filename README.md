# AI Nepali ↔ Sinhala Translator

Compact, developer-friendly README prepared for repository push.

A lightweight web app for extracting and translating Nepali and Sinhala text from images and PDFs. This repo contains the frontend (Vite + React + TypeScript) and tooling for local development.

**Status:** Prototype / Hackathon build

**License:** MIT

--

## Features

- OCR for Nepali (Devanagari) and Sinhala
- Neural translation to English with editable output
- Drag-and-drop upload, PDF & image support
- Offline-capable frontend and local exports

--

## Tech stack

- Frontend: React, TypeScript, Vite, Tailwind CSS
- Tooling: npm scripts (dev, build, preview)

--

## Quick start

Prerequisites: Node.js 16+ and npm.

```bash
# clone
git clone <your-repo-url>
cd AI-Nepali-Sinhala-Translator

# install
npm install

# development server
npm run dev

# build for production
npm run build

# preview production build
npm run preview
```
